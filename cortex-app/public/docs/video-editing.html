Video Editing

Edit video in DaVinci Resolve, Premiere Pro, and CapCut using natural language. CORTEX automates timeline operations, exports, and effects — all from a single prompt.

Overview

CORTEX brings natural language video editing to your desktop. Describe the edit you want in plain English — "cut the clip at 10 seconds and add a cross dissolve" — and CORTEX translates your intent into editor-specific commands. There is no need to memorise hotkeys, dig through menus, or write scripts.

The system works with the three most popular professional and consumer editing applications on the platform. Each editor is driven through keyboard shortcut simulation (hotkeys). CORTEX sends key combinations to trigger common editing commands.

Supported Editors

CORTEX interfaces directly with the three most popular video editing applications:

  • DaVinci Resolve — automated via keyboard shortcut simulation (hotkeys)
  • Premiere Pro — automated via keyboard shortcut simulation (hotkeys). CORTEX sends key combinations to trigger common editing commands.
  • CapCut — automated via keyboard shortcut simulation (hotkeys)

What You Can Do

Each editor exposes a rich set of automatable operations. The table below outlines the capabilities available per editor.

DaVinci Resolve

  • Create, open, and save projects
  • Add, move, trim, and delete clips on the timeline
  • Apply transitions, effects, and color grades
  • Add text and shape titles
  • Set in/out points and render exports

Premiere Pro

  • Create sequences and import media
  • Add clips to timeline with snap control
  • Apply video and audio transitions
  • Adjust clip speed, scale, and position
  • Export via Media Encoder presets

CapCut

  • Import media and create projects
  • Split, trim, and reorder clips
  • Apply auto-captions and text overlays
  • Adjust volume, speed, and keyframes
  • Export to local file or TikTok

Prerequisites

Before using the video editing feature, ensure the following:

  • The video editing application is installed and logged in (if required)
  • For DaVinci Resolve: ensure the application is installed and running
  • For Premiere Pro: ensure the application is installed and running
  • For CapCut: ensure the application is installed and running
Note: Video editing automation requires the Automation permission level. Grant it in the Permissions panel before issuing any editing commands. Without this permission, CORTEX will read the timeline state but will not make changes.

Example Workflow

The following walkthrough shows a typical editing session using DaVinci Resolve. The same pattern applies to Premiere Pro and CapCut.

1. Open or create a project

$ "open my latest DaVinci Resolve project"

If no project is specified, CORTEX creates a new one with a timestamp-based name.

2. Import media

$ "import the clips from ~/Downloads/gameplay into the media pool"

CORTEX can also pull media from Finder selections, browser downloads, or a specific folder path.

3. Build the timeline

$ "add clip_01.mov to the timeline, then trim the first 3 seconds off"
$ "append the rest of the clips in order"

4. Apply effects and transitions

$ "add a cross dissolve between clip 1 and clip 2"
$ "apply the 'Film Looks' LUT to the entire timeline"

5. Add titles

$ "add a title overlay that says 'Level 1 Complete' in the top-left corner"

6. Export

$ "export the timeline as H.264, 1080p, 30fps"

CORTEX sets the render preset, configures the output path, and starts the render. You are notified when the export finishes.

Tip: Editor-specific commands are most reliable when you include the editor name in your prompt (e.g., "in DaVinci Resolve, apply a color grade to clip 2").
