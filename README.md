# AURA

[![GitHub](https://img.shields.io/badge/GitHub-AURA--v1-7c5cfc)](https://github.com/mithunsuresh121-stack/Aura-v1)

Your local AI agent that runs entirely on your machine.

**GitHub:** https://github.com/mithunsuresh121-stack/Aura-v1

## Why AURA?

- **Private.** Everything runs locally. No servers, no logs, no data collection.
- **Free.** No GPU required. No subscriptions, no tokens, no usage caps.
- **Capable.** Multi-model chat, computer control, terminal sessions, video editing, MCP connectors, and agent orchestration.

## Getting Started

```bash
git clone https://github.com/mithunsuresh121-stack/Aura-v1.git
cd Aura-v1/autonomous
bash start.sh
```

Then open `http://localhost:5173` in your browser.

## Capabilities

- **Multi-Model Chat:** Switch between Ollama, OpenAI, Gemini at runtime. Agent mode for tool use.
- **Computer Control:** Screen capture, click, type, drag, run scripts — platform-native on macOS, Windows, Linux.
- **Terminal:** Persistent PTY shell sessions for git, package management, dev servers, and scripting.
- **Video Editing:** Automate DaVinci Resolve, Premiere Pro, and CapCut through natural language.
- **MCP Connectors:** Plug into GitHub, Drive, Slack, and any MCP-compatible service.
- **Agent Orchestration:** Decompose complex tasks into sub-agents. Plan, delegate, review.
- **Privacy-First Security:** Granular permissions per capability. No telemetry. No cloud dependency.
