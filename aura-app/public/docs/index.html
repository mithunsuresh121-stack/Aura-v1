Quick Start

Clone the repository and start the server:

$ git clone https://github.com/mithunsuresh121-stack/Aura-v1.git
$ cd Aura-v1/autonomous
$ bash start.sh

Open http://localhost:5173 in your browser. The sidebar shows connection status and available capabilities.

Tip: No API keys required. AURA defaults to Ollama if available, with GPT-2 fallback. For cloud models, configure OpenAI or Gemini in the Settings panel.

Architecture

AURA is split into two components:

  • Server — Python backend with Ollama bridge (default), distilled checkpoint, and GPT-2 fallback. Capability plugins include computer control, terminal, MCP, orchestration, and video editing. Exposes a REST API on :8081.
  • App — Vue 3 frontend with a premium glassmorphism UI. Connects to the server and provides the chat interface, settings, and build panel.

Privacy: Everything runs locally. No data leaves your machine unless you explicitly connect to a cloud model.
